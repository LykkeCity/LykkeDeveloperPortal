Getting Started with the Lykke Exchange Public API

Information from our exchange is available via API. No key is needed, but requests are rate-limited to 60 per minute. More details and a full list of endpoints are available in the API Docs.

Fetching Basic Data

Let's start by getting the current BTC / USD exchange rate.

To perform the HTTP requests, the examples will use node-fetch, but any request library will do. In this instance, we want to issue a GET request to the AssetPairs/rate endpoint, appending the asset pair ID to fetch the BTC / USD rate. 


  const fetch = require("node-fetch");

  // URL format is https://public-api.lykke.com/api/AssetPairs/rate/{assetPairId}
  // For full API details, visit https://public-api.lykke.com/
  const url = "https://public-api.lykke.com/api/AssetPairs/rate/BTCUSD";

  const getAssetPairRate = async url => {
    try {
      const response = await fetch(url);
      const json = await response.json();
      console.log(json);
    } catch (error) {
      console.log(error);
    }
  };

  const assetPairRateData = getAssetPairRate(url);
