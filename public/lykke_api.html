How to get API Key

  1. Use the following link to create API Wallet:
    https://wallet.lykke.com/wallets/hft
  2. Create a new API Wallet and save provided API Key securely
  3. Deposit funds from the Trading wallet:
    https://wallet.lykke.com/transfer

  4. Confirm transfer using the LykkeWallet app

    IOS 8+ and Android 4.4.2+ required
