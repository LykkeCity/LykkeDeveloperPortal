Getting Started with the Lykke HFT API

Interested in high-frequency or programmatic trading on the Lykke Exchange? These steps will get you going, then you can explore all the available functionality in our HFT API Docs. The API is currently in the beta testing stage, and throughput is limited to 300 requests/min.

Getting an API Key

  1. Use the following link to create an API Wallet:

    https://wallet.lykke.com/wallets/hft

  2. Create a new API Wallet and save your provided API Key in a secure place

  3. Deposit funds from the Trading wallet into your API Wallet:

    https://wallet.lykke.com/transfer

  4. Confirm the transfer using the LykkeWallet app

    IOS 8+ and Android 4.4.2+ required

  5. Trade!
